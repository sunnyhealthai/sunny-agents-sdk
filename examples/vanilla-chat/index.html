Sunny Agents SDK Example

Start the dev server with npm run dev, then optionally paste an access token to talk to the real Sunny backend. Leaving the token blank keeps the widget in anonymous/local mode.

Authentication Options

SDK Configuration

import { createSunnyChat } from '@sunnyhealthai/agents-sdk';

const chat = await createSunnyChat({
  container: document.getElementById('chat'),
  partnerIdentifier: 'guardian-mock',
  publicKey: 'YOUR_PUBLIC_KEY',
  authType: 'passwordless',
});

// Clean up when unmounting
// chat.destroy();
Enter your email or phone number to receive a verification code. After verification, you'll be automatically authenticated.

SDK Configuration

import { createSunnyChat } from '@sunnyhealthai/agents-sdk';

const chat = await createSunnyChat({
  container: document.getElementById('chat'),
  partnerIdentifier: 'guardian-mock',
  publicKey: 'YOUR_PUBLIC_KEY', // from API Keys section below
  authType: 'tokenExchange',
  idTokenProvider: async () => {
    // Return your user's current ID token
    return localStorage.getItem('id_token');
  },
});

// Clean up when unmounting
// chat.destroy();
Paste an ID token from your identity provider. The SDK will exchange it for an access token automatically. Tokens are stored in memory only and will be lost on page reload.
